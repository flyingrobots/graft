# Structural churn report

Which symbols change most frequently? High-churn symbols are
maintenance hotspots. Count `changes` edges per symbol across
the worldline.

Output: ranked list of symbols by change frequency, with files
and last-changed commit. Pairs with symbol-heatmap for
visualization.

Depends on: WARP Level 1 (shipped).

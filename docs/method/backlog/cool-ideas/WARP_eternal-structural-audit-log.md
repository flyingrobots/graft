# Eternal structural audit log

Every structural observation is recorded. Every projection choice.
Every alternative NOT shown. The full multiverse of what the agent
could have seen vs what it did see.

The EAL (from AION Paper 19) generalizes git history, AST diffs,
execution traces, and audit trails into a single unified object.
Applied to graft: the structural observation history of every
agent session, compressed via structure sharing.

"Why did the agent introduce this bug?" Replay its observation
DAG. It never saw the updated signature because it read from
cache. The EAL proves it.

Self-compressing: structural universes share state. The audit
log does not grow exponentially.

Depends on: WARP Level 1 (shipped), provenance DAG (backlog),
agent action provenance (backlog).

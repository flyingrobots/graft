# Graveyard

This directory holds backlog items and related METHOD notes that should remain
searchable but are no longer live work because they failed the relevance check.

Completed work does not go here. For relevant work, move the backlog card to
`docs/design/<task>.md`, run the cycle, and record completion in
`docs/method/retro/` or `docs/releases/<version>/retros/`.

Use the graveyard only when a proposal is canceled, superseded, absorbed as
standalone work by another cycle, or no longer relevant. Entries should explain
their disposition. `docs/method/backlog/` is for live backlog lanes only;
`docs/method/backlog/graveyard/` is not a valid completion or retirement
location.

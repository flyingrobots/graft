# Graveyard

This directory holds retired backlog items and related METHOD notes
that should remain searchable but are no longer live work.

Completed work does not go here. Completion is represented by a closed
cycle packet: design, witness, and retro under `docs/method/retro/` or
`docs/releases/<version>/retros/`.

Use the graveyard only when a proposal is canceled, superseded,
absorbed as standalone work by another cycle, or no longer relevant.
Entries should explain their disposition. `docs/method/backlog/` is for
live backlog lanes only; `docs/method/backlog/graveyard/` is not a
valid completion or retirement location.

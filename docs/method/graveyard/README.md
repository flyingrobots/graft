# Graveyard

This directory holds retired backlog cards and related METHOD notes
that should remain searchable but are no longer live work.

Never delete a backlog card outright. When a card leaves
`docs/method/backlog/`, move the card here with a disposition note so
the original backlog record remains searchable.

Completion is additionally represented by a closed cycle packet:
design, witness, and retro under `docs/method/retro/` or
`docs/releases/<version>/retros/`. The retro proves completion; the
graveyard preserves the retired card.

Use the graveyard when a proposal is completed, canceled, superseded,
absorbed as standalone work by another cycle, or no longer relevant.
Entries should explain their disposition. `docs/method/backlog/` is for
live backlog lanes only; `docs/method/backlog/graveyard/` is not a
valid completion or retirement location.

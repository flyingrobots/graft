# v0.2.2

## Summary

Docker support and canonical JSON. Run graft without installing Node,
and all JSON output is now deterministic with sorted keys.

## What Changed

- **Docker support**: run graft as an MCP server in Docker:
  ```bash
  docker run -i --rm -v "$PWD:/workspace" flyingrobots/graft
  ```
  Node 22 Alpine, non-root user, works with any MCP client.
- **Canonical JSON codec**: all JSON serialization uses sorted keys
  and compact output. Deterministic: same data always produces the
  same string. Enables stable hashes and diffable logs.

## Why It Matters

Docker removes the Node.js prerequisite for teams that want graft
without managing a Node installation. Canonical JSON ensures
receipts and metrics logs are reproducible and diffable.

## Breaking Changes

None.

## Migration

No migration required. Docker is a new deployment option alongside
the existing `npx @flyingrobots/graft`.
